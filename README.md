# AdaptFedDS
This code implements a lightweight dual-reinforcement learning method for optimal device selection in learning processes , optimizing participation levels and communication rounds to enhance validation accuracy while minimizing the total number of rounds needed.

Article link: https://link.springer.com/article/10.1007/s10586-025-05765-4

## Requirements
AdaptLightFed-Ds has been implemented and tested with the following versions: 
- Python (v3.11.3).
- Pytorch (v2.0.0).
- Scikit-Learn (v1.2.2).
- Scipy (v1.10.1).
- FedLab (v1.3.0).
- NumPy (v1.24.3).
